Event binding in angular

In angular, all the events are executed in bubbling phase. Angular does not provide any support to capture phase. If we want to use the capture phase we have to write our custom event handling logic. We can do this with the help of "custom directives"

If we want to access, the event object for an element we can use $event, which helps use use the browser's native event object. Angular makes sure that the events work properly in each and every browser with the help of @angular/platform-browser
